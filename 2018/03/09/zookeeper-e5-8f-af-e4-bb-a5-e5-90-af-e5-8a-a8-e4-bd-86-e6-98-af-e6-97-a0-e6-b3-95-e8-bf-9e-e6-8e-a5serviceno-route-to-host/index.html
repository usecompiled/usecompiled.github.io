(Zookeeper可以启动但是无法连接Service)No route to host

启动server只需运行脚本: bin/zkServer.sh start 这里可以正常执行。 Server启动之后, 就可以启动client连接server了, 执行脚本: bin/zkCli.sh -server localhost:2181 但是执行上面的连接时一直报错(报错信息如下)，网上找了半天问题，最后发现把防火墙关了就OK啦！ WARN [main-SendThread(192.168.1.121:2181):ClientCnxn$SendThread@1162] - Session 0x0 for server null, unexpected error, closing socket connection and attempting reconnect java.net.NoRouteToHostException: No route to host

chkconfig iptables off
systemctl stop firewalld.service #停止firewall
systemctl disable firewalld.service #禁止firewall开机启动
