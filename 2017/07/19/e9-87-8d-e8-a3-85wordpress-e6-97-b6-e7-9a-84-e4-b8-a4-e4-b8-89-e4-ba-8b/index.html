重装wordpress时的两三事

最近在折腾网站，想换typecho，未遂，后来又重装了wordpress，结果在安装的时候出现了几个问题：

  1. 在填写数据库信息时 提示数据库连接错误
  2. 提交之后提示500 Internal Server Error
  3. 在设置“使用主域名访问blog”时之后 无法访问

三个问题的解决方案依次为：

  1. 在确定数据库名，用户名，密码填写无误后，经查询得知，使用万网主机，所要填写的主机名为提供的数据库连接地址。
  2. 万网主机 出现此情况，将主机的php版本改为php5.4以上即可。
  3. 将.htaccess文件以及index.php文件复制至网站服务器的根目录中，问题解决。当然也可以移动或上传所有解压后的WordPress文件夹里面的文件(但不包括WordPress文件夹本身) 到你服务器的根目录下
