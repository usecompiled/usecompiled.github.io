android学习

1. Android Studio 默认编辑器(Editor)的方android studio下的java与android学习：案(Scheme)是无法修改字体的, 可以Save as, 保存为新的方案(Scheme), 然后更改字体大小; 位置: File->Settings->IDE Settings->Editor->Colors & Fonts -> Font —————————————————————————— 2. 调用AVD虚拟机时报错如下： emulator: ERROR: x86 emulation currently requires hardware acceleration Please ensure Intel HAXM is properly installed and usable. CPU acceleration status: HAX kernel module is not installed! http://stackoverflow.com/questions/26355645/error-in-launching-avd http://stackoverflow.com/questions/27720064/hax-kernel-module-is-not-installed 经过STACK 以及 google 发现，出现此告警的原因是Avd调用X86架构的安卓虚拟机需要使用到Intel HAXM引擎，而本机尚未进行安装导致。 HAXM下载 HAXM的安装文件在Android SDK中是默认包含的，文件路径位于 C:\Users\XXXX\AppData\Local\Android\sdk\extras\intel\Hardware_Accelerated_Execution_Manager\intelhaxm-android.exe 具体路径根据本机Android SDK的安装路径不同可能会有所区别。 也可以从Intel官网上进行下载，地址为：https://software.intel.com/en-us/android/articles/intel-hardware-accelerated-execution-manager CPU如果是AMD的话，HAXM肯定是不支持的，不支持虚拟化（VT）和eXecute Disable bit（XD）的话肯定也是无法安装HAXM的。 此时进入bios： 1.打开vt 将改为 enabled 2.打开xd ———————————————————————————————- 3. Android Studio真机调试 要在Android Studio中每次运行程序(run)都可以在手机中直接看到效果，需要执行以下几个步骤： 1.使用USB线连接你的手机，并在手机中开启”USB调试”。 2.在Android Studio中，点击菜单”Run - Edit Configurations”。 3.在General选项卡中，将”Target Device”的选项改为”USB device”，点ok确定。 4.完成！再次按ctrl+F10 “run”一次，如果你发现软件下方区域出现类似这样的回馈 “Installing com.example.android.BluetoothChat DEVICE SHELL COMMAND: pm install -r “/data/local/tmp/com.example.android.BluetoothChat” pkg: /data/local/tmp/com.example.android.BluetoothChat Success” —————————————————————————————- 4. https://developer.android.com/sdk/installing/index.html?pkg=tools 1.环境变量”，在“系统变量”中新建SDK_HOME变量，并将SDK目录添加进去 2.编辑Path变量。保证现有变量值后面有一个分号“;”，然后添加%SDK_HOME%\tools;%SDK_HOME%\platform-tools; 3.SDK环境配置完毕。 测试是否成功：开始-运行-输入cmd-在终端输入 android -h —————————————————————————————
